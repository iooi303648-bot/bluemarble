# 국기 이미지 넣는 방법

이 폴더에 아래 파일 이름으로 PNG 이미지를 넣으면 보드판의 나라 칸에 색상 아이콘 대신 실제 국기 사진이 자동으로 표시됩니다. 이미지를 넣지 않아도 각 나라 색상으로 만든 아이콘이 항상 안정적으로 보이니(인터넷 연결이나 폰트에 상관없이 동작) 안 넣어도 게임은 정상 동작합니다.

| 나라 | 파일 이름 |
| --- | --- |
| 대한민국 | `kr.png` |
| 일본 | `jp.png` |
| 중국 | `cn.png` |
| 사우디아라비아 | `sa.png` |
| 이집트 | `eg.png` |
| 케냐 | `ke.png` |
| 프랑스 | `fr.png` |
| 영국 | `gb.png` |
| 미국 | `us.png` |
| 캐나다 | `ca.png` |
| 브라질 | `br.png` |
| 오스트레일리아 | `au.png` |

권장 크기는 가로 96px × 세로 72px 정도의 작은 국기 이미지(PNG)입니다. 파일을 넣고 저장하면 새로고침만으로 바로 반영됩니다.
